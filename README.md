## Function 說明

這個題目是做一個簡易的答題系統，題目與答案存在後端的 MongoDB，前端在開啟或是 reload 的時候會把題目透過 axios (RESTful APIs) 從後端資料庫取得。

使用者可以透過前端點選答案，並按「next」進入下一題，在全部題目作答完畢之後，後端會比對答案，並計算成績，呈現在螢幕上。
